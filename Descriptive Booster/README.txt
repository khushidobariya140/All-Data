:-:-:- Household Income Distribution Analysis :-:-:-

-> Project Overview :-
This project focuses on analyzing household income distribution using Python.
The dataset contains 150+ records generated in a normal distribution format and
is used to perform statistical analysis and visualization.

:-The goal of this project is to:

Analyze income distribution patterns

Check normality of the dataset

Calculate statistical measures

Visualize distribution trends

Perform business-level insights

:- Objectives :-
Generate normally distributed household income data

Calculate descriptive statistics

Perform distribution analysis

Identify skewness and kurtosis

Visualize data using histograms and probability density curves

Interpret insights for business case study

:- Dataset Description
The dataset contains the following columns:

Household_ID

Household_Size

Monthly_Income

Annual_Income

Region

Employment_Type

Education_Level

-> Total Records: 150+
-> Distribution Type: Normal Distribution

---- Statistical Analysis Performed
Mean

Median

Standard Deviation

Variance

Skewness

Kurtosis

Normal Distribution Curve

Histogram Visualization

---- Visualizations Included

Histogram of Monthly Income

Normal Distribution Curve

Income Distribution Plot

Statistical Summary Table

:-:-:-:- Key Insights
Income data follows approximately normal distribution.

Slight positive/negative skewness observed (if applicable).

Majority of households fall within middle-income range.

Outliers (if any) identified using statistical measures.
