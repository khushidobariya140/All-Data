#  Data Analysis Project README

## Overview

This project demonstrates the complete data analysis pipeline
including: - Data Acquisition - Data Cleaning - Exploratory Data
Analysis (EDA) - Data Profiling

------------------------------------------------------------------------

## Part B: Data Acquisition

-   Loaded dataset from CSV using Pandas.
-   Parsed JSON file into DataFrame.
-   Merged multiple datasets.

------------------------------------------------------------------------

## Part C: Data Understanding & Cleaning

-   Explored dataset using head(), info(), describe().
-   Identified missing values and duplicates.
-   Handled missing values using mean/median.
-   Removed duplicate rows.
-   Verified data types.

------------------------------------------------------------------------

## Part D: Exploratory Data Analysis (EDA)

### Univariate Analysis

-   Distribution of Age
-   Distribution of Income

###  Bivariate Analysis

-   Gender vs Purchases
-   Income vs Churn

###  Multivariate Analysis

-   Correlation Heatmap
-   Pair Plot

------------------------------------------------------------------------

## Part E: Data Profiling

-   Generated automated report using ydata-profiling.
-   Included missing values, correlations, and warnings.

------------------------------------------------------------------------
