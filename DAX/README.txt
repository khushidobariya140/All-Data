Overview:-

This project demonstrates Power BI data modeling and DAX concepts using sales and returns data. 
It includes calculated columns, measures, time intelligence, and matrix-based reporting.

Dataset:-

Sales Fact,
Returns Fact,
Customer, Product, Date, Region Dimensions

Key Features:-

Calculated columns (Profit, Return Flag, Customer Full Name)

DAX measures for Sales, Cost, Profit, and Returns

Time intelligence (YoY growth, monthly comparison)

Filter context handling using CALCULATE, ALL, FILTER

Matrix visual showing Sales by Region and Month (no slicers)

Conclusion:-

This project helped in understanding practical DAX usage,filter context behavior,
and time intelligence functions while building a clean and insightful Power BI report.
