------------------------ Project Title ------------------------

:- Health Data Analysis using Statistical Methods in Python

-> Project Description:

This project focuses on performing statistical analysis on a healthcare dataset containing 250 records.
The dataset includes variables such as age, weight, BMI, smoking status, diabetes, and other health indicators.
The objective is to apply inferential statistical techniques to extract meaningful insights and validate hypotheses.

-> Dataset Information:

The dataset consists of the following attributes:

- Record ID (Unique identifier)
- Age & Age Group
- Weight
- Gender & Region
- Smoking Status
- Exercise Frequency
- BMI (Body Mass Index)
- Blood Pressure
- Diabetes & Hypertension
- Cholesterol Level
- Glucose Level
- Visit Date

The data is synthetically generated but follows realistic distributions.

-> Objectives:
To understand and apply statistical concepts using real-like data
To perform hypothesis testing and interpret results
To analyze relationships between health variables
To make data-driven conclusions

-> Tools & Technologies Used:
Python
Libraries:
Pandas
NumPy
SciPy

-> Tasks Performed:

:- Hypothesis Formulation -:

Defined null and alternative hypotheses for statistical testing
Confidence Interval Calculation
Calculated 95% confidence intervals for age and weight.

:- T-Test & Z-Test -:

Compared means across groups
Tested statistical significance
Chi-Square Test
Analyzed relationship between categorical variables (Smoking vs Diabetes).

:- ANOVA Test -:

Checked if age groups differ significantly in disease occurrence
Covariance & Correlation
Measured relationship between age and BMI
Result Interpretation
Accepted or rejected hypotheses based on p-values.

-> Key Findings:

- Smoking does not significantly affect BMI.
- Smoking and diabetes are statistically independent.
- Age differs significantly from the assumed population mean.
- No strong variation in disease rate across age groups.
- Weak positive correlation exists between age and BMI.

-> Conclusion:

This project demonstrates how statistical methods can be used to analyze healthcare data effectively.
It helps in understanding relationships between variables and supports decision-making based on data.
